# 🌸 Cycle Bloom

A mobile-first Period Tracker PWA built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Onboarding screen
- Last period date input
- Cycle length input
- Period duration input
- Next period prediction
- Ovulation estimation
- Calendar visualization
- LocalStorage persistence

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- localStorage

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open `http://localhost:3000` in your browser.

## Build

```bash
pnpm build
```
